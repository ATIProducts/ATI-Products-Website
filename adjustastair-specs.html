FAQ

Technical Questions Answered

What materials is AdjustaStairs made from?

AdjustaStairs uses lightweight aircraft-grade aluminum — durable for daily job site use while light enough for one or two workers. ATI Construction Products engineers every system to balance job site demands with OSHA safety requirements.

How many workers are needed to install AdjustaStairs?

AdjustaStairs installs with one or two workers and no tools required. The tool-free adjustment mechanism allows quick height changes and secure locking, making it fast to deploy and redeploy across multiple project locations.

Can AdjustaStairs be reused across multiple job sites?

Yes. AdjustaStairs is a portable temporary system designed to move with the crew from project to project. Its adjustable height adapts to different roof elevations and site conditions across all project types — commercial, residential, and industrial.

How does AdjustaStairs compare to a ladder?

OSHA 1926.1051 requires stairways when elevation changes exceed 19 inches and the task requires frequent access. AdjustaStairs provides a safer, OSHA-compliant alternative that reduces fall risk, allows workers to carry materials safely, and improves site productivity.
